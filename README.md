# Tweeter Project

Tweeter is a simple, single-page Twitter clone. It allows students to practice their HTML, CSS, JS, jQuery and AJAX front-end skills.

## Dependencies

- Express
- Node 5.10.x or above
- Body-parser
- Chance
- Normalize.css
